# Habit Tracker

A simple and effective web application to help users build, track, and maintain positive habits.  
This project allows users to create custom habits, monitor progress, and visualize streaks to stay motivated.

## Features
- Add, edit, and delete habits
- Daily/weekly progress tracking
- Visual streak counter
- Responsive design for desktop and mobile
- Clean and minimal UI

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sharminsheik47-wq/Habit-tracker.git
2. Navigate into the project folder:
   ```bash
   cd Habit-tracker
3. Install dependencies:
   ```bash
   npm install
4. Run the project:
   ```bash
   npm start

## Usage
- Add new habits with custom names and schedules.
- Mark habits as complete each day to maintain streaks.
- View progress charts to stay motivated.
